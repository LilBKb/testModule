# Модуль "Управление заявками"

Тестовое задание для фронтенд-разработчика. Модуль для управления заявками с возможностью фильтрации, просмотра и редактирования.

## Технологии

- React 18.2 (hooks)
- React Router v6
- Ant Design 5.12
- Axios
- SCSS
- Vite
- Day.js

## Установка и запуск

### Требования

- Node.js 16.x или выше
- npm или yarn

### Запуск проекта

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Приложение откроется автоматически по адресу: http://localhost:3000

### Дополнительные команды

```bash
npm run build   # Сборка для production
npm run preview # Просмотр production сборки
```

## Примечания

⚠️ **Mock-данные**: В проекте используются mock-данные вместо реального API. Все данные находятся в `src/api/mockData.js` и хранятся в памяти (сбрасываются при перезагрузке).

## Структура проекта

```
src/
├── api/          # API слой и mock-данные
├── components/   # Переиспользуемые компоненты
├── pages/        # Страницы приложения
├── hooks/        # Custom hooks
└── styles/       # SCSS стили
```

## Маршруты

- `/` - Список заявок
- `/application/:id` - Карточка заявки
